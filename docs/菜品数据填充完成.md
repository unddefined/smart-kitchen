## 🚀 使用方法

### 本地开发环境

1. **配置环境变量**
   ```bash
   # 在 backend/.env 文件中设置
   DATABASE_URL=postgresql://smart_kitchen_user:密码@localhost:5432/smart_kitchen_prod
   ```

2. **安装依赖**
   ```bash
   cd backend
   npm install
   ```

3. **生成 Prisma Client**
   ```bash
   npx prisma generate
   ```

4. **运行种子脚本**
   ```bash
   npm run seed:dishes
   ```

### 生产环境（服务器）

**方法一：使用 SQL 脚本**
```bash
# SSH 登录服务器
ssh -i "C:/Users/66948/.ssh/PWA应用密钥.pem" root@8.145.34.30

# 进入项目目录
cd /path/to/smart-kitchen

# 执行 SQL 脚本
psql -h localhost -U smart_kitchen_user -d smart_kitchen_prod \
  -f database/insert-dishes.sql
```

**方法二：使用 Bash 脚本**
```bash
# 添加执行权限（仅首次）
chmod +x scripts/init-dishes-data.sh

# 运行脚本
./scripts/init-dishes-data.sh
```

## 🔒 数据安全保证

根据**生产环境数据安全规范**，本种子脚本遵循以下最佳实践:

✅ **增量式设计**: 只添加新数据，不修改或删除现有数据  
✅ **幂等性检查**: 使用 upsert 操作，自动跳过已存在的记录  
✅ **无破坏性操作**: 不包含 deleteMany、truncate 等危险操作  
✅ **详细日志**: 所有操作都有明确的日志输出  
✅ **可追溯**: 通过 git 提交记录追踪变更
